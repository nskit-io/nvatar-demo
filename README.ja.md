## NVatarの特徴

ほとんどのAIチャットボットはステートレスなテキスト生成器です。NVatarは**3Dルームに住むAIの友達** — 会話を記憶し、時間とともに性格が発展し、9次元の感情を追跡し、クローンされた人間の声で32以上の言語を話します。

### ハイブリッドAIアーキテクチャ
**ローカル**と**クラウド**AIの長所を融合：
- **ローカルLLM**（Gemma 26B MoE、Apple Silicon）— 性格、会話、感情。高速でプライベート。
- **クラウド検索**（Claude WebSearch）— 事実の回答、リアルタイムデータ。必要な時だけ。

個人的な会話を外部サーバーに送ることなく、感情的な会話（ローカル）も今日の為替レート検索（クラウド）も可能です。

### 3層メモリシステム
アバターは単に応答するのではなく、**記憶**します：
- **L1（原本）** — 完全な会話履歴
- **L2（要約）** — 重要な瞬間を関連性スコアとともに圧縮
- **L3（キーワード）** — 永続的な長期性格キーワード

100メッセージを超えると、L1が自動的にL2/L3に圧縮 — 人間が詳細よりも要点を覚えるように。

### 9次元感情トラッキング
メッセージごとに8軸の感情状態が更新：
`喜び · 悲しみ · 怒り · 不安 · 愛情 · 興奮 · 退屈 · 信頼`

感情は時間とともに自然に減衰し、高い感情は3Dアバターの表情やジェスチャーとして現れます。

### 性格進化
基本性格（フレンドリー、穏やか、ツンデレなど）＋ランダムMBTIからスタート。会話が蓄積されると、8つの性格特性 + MBTI 4次元が`pending_delta → decay → commit`サイクルで変化します。

### 音声クローニングTTS
ロボット音声ではなく、**クローンされた人間の声**で32以上の言語を自然に：
- 言語別速度自動調整（韓国語：0.85x、日本語：0.65x）
- テキスト前処理（絵文字除去、発音ガイド除去）
- キューベース順次再生＋入力時即座中断

### ソーシャルエコシステム — 「人格は1つ、関係はたくさん」
多くのAIコンパニオンは1:1に閉じています。NVatarは**自分の他のアバターを同じ部屋に招待して、彼らが互いに関係を築くように**します。

- **人格はユーザーとの会話からのみ**形成 — アバター同士の対話は人格に影響しない
- **関係はペア別に記録** — 親密度・会話回数・最近の話題が `nv_avatar_relationships` に蓄積
- **メモリトラック分離** — A↔Bの会話コンテキストはA↔ユーザーのコンテキストを汚染しない
- **Room Manager** — 中央スケジューラが発話タイミング、キュー、sticky target、auto-cascade(最大2ターン)を調整
- **文脈に合った口調** — ユーザーには設定した敬語レベル、アバター同士はタメ口
- **AFK安全** — タブ非表示で対話停止、1時間あたり最大6回キャップ(スクリーンセーバー用途にも対応)

「👥 友達」パネルで自分の別アバターを招待すると8秒以内に挨拶が交わされ、名前を呼べばチェーン会話が自然に展開します。

## 主な機能

| 機能 | 説明 |
|------|------|
| **3Dバーチャルルーム** | VRMアバター + Mixamo 33種アニメーション |
| **自然な会話** | Gemma 26B MoE + 性格・記憶・感情 |
| **音声出力** | ElevenLabs Voice Clone — 32言語以上 |
| **音声入力** | Whisper STT — 自動言語検出 |
| **ウェブ検索** | リアルタイム事実検索 + 構造化結果 |
| **多言語** | KO、JA、EN、ZH — UIと会話 |
| **ソーシャルエコシステム** | 自分のアバター同士を会わせる — ペア別親密度、人格は独立 |
| **Room Manager** | 複数アバター対話の中央調整 — キュー / sticky / auto-cascade |
| **Room Authoring** | Edit モードでシーンのメッシュを複数選択 → グループ化 → DB 保存 |

## はじめ方

1. **[ライブデモ](https://nskit-io.github.io/nvatar-demo/)** を開く
2. **キャラクターモデル**選択 → **名前**入力 → **性格・言語**選択
3. **Create Avatar** → ルーム入場！

### ルーム操作

| 機能 | 使い方 |
|------|--------|
| チャット | 入力 → 送信（Enter） |
| 音声入力 | 🎤 → 録音 → 自動認識 → 自動送信 |
| アバター移動 | 床をダブルクリック |
| 言語変更 | サイドパネル（会話リセット） |
| TTS オン/オフ | サイドパネル 🔊 |
| 友達招待 | トップバー → 👥 友達 → 自分の別のアバターを [入場] |
| 特定の友達に呼びかけ | "ルビンちゃん、こんにちは" — 名前プレフィックスでルーティング |
| ルーム編集 | トップバー → ✏️ Edit → ギズモ移動 + ✍ Authoring パネル |
| パネル移動 | ヘッダーをドラッグ / ダブルクリックで折りたたみ |
| 検索結果 | バッジクリック → 閲覧 |
