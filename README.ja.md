# NVatar デモ — AI アバターボイスチャット + 3D バーチャルルーム

[English](README.md) | [한국어](README.ko.md)

3Dバーチャルルームで、AIアバターの友達に会いましょう。自然な会話、声のクローニングTTS、リアルタイムの感情表現とジェスチャーを体験できます。

**[ライブデモを試す](https://nskit-io.github.io/nvatar-demo/)**

## 主な機能

- **3Dバーチャルルーム** — VRMアバター + Mixamo 33種アニメーション（待機、歩行、感情、ジェスチャー、ダンス）
- **自然な会話** — Gemma 26B MoE搭載、性格・記憶・感情トラッキング
- **音声出力** — ElevenLabs Voice Clone TTS（韓/日/英/中/西/仏 + 32言語）
- **音声入力** — Whisper STT（自動言語検出）
- **ウェブ検索** — リアルタイム事実検索 + 構造化された結果
- **多言語対応** — 韓国語、日本語、英語、中国語 — UIと会話の両方
- **アバターラボ** — VRMモデル + Mixamo FBXアニメーションテスト

## はじめ方

1. **[ライブデモ](https://nskit-io.github.io/nvatar-demo/)** を開く
2. グリッドから**キャラクターモデル**を選択
3. アバターの**名前**を入力
4. **性格**と**言語**を選択
5. **Create Avatar**をクリック → ルームに入場！

### ルーム内の操作

| 機能 | 使い方 |
|------|--------|
| **チャット** | 入力欄にメッセージを入力 → 送信（またはEnter） |
| **音声入力** | 🎤ボタンをクリック → 録音 → 自動認識 → 自動送信 |
| **アバター移動** | 床をダブルクリック |
| **言語変更** | サイドパネル → 言語セレクター（会話がリセットされます） |
| **TTS オン/オフ** | サイドパネル → 🔊 TTSトグル |
| **家具配置** | サイドパネル → デスク、棚、ランプ、植物 |
| **検索結果** | アバターが検索するとバッジ表示 → クリックして閲覧 |

### ヒント

- アバターは**会話内容を記憶**し、時間とともに性格が発展します
- 「今日の為替レートは？」のような事実の質問をすると検索してくれます
- 会話中に言語を切り替えても、アバターは自然に適応します
- ユーザーあたり最大**3体のアバター** — 古いものを削除して新しく作成できます

## 技術スタック

| コンポーネント | 技術 |
|-------------|------|
| AIモデル | Gemma 4 26B MoE (4-bit, Apple Silicon MLX) |
| TTS | ElevenLabs Voice Clone (turbo v2.5) |
| STT | Whisper large-v3 (MLX) |
| 3D | Three.js + @pixiv/three-vrm + Mixamo FBX |
| 検索 | CSW (Claude) WebSearch ハイブリッド |
| バックエンド | Python FastAPI + WebSocket |
